h1. Auto da Compadecida Game
========================

Jogo desenvolvido pelo pessoal da célula de jogos que usam a linguagem Python na cidade de Caruaru em 
Pernambuco. Este projeto foi desenvolvido para uma disciplina do curso de Sistemas de Informação da 
Universidade de Penambuco - UPE.

O projeto foi apresentado no VI Encontro do Grupo de Usuários Python de Pernambuco "PUG-PE":http://pugpe.wordpress.com.  
O Resumo do encontro você encontra "aqui":http://pugpe.wordpress.com/2010/08/16/resumo-vi-encontro-pugpe/
A palestra na íntegra está disponível também em "vídeo!":http://pug-pe.blip.tv/
